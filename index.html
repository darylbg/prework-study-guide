Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The footer element represents a footer for its nearest ancestor sectioning content or sectioning root element.

CSS

  • CSS stands for Cascading Style Sheets
  • CSS is a language for specifying how documents are presented to users
  • CSS specifies how documents are styled, laid out, etc...

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it

JavaScript

  • JavaScript is a scripting language.
  • JavaScript enables you to create dynamically updating content, control media and images
  • JavaScript is the most used programming language in the world.
